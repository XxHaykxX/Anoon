# anoon-admin — Design System

> Источник: ui-ux-pro-max (движок) + референс Datta Able (layout) + бренд app (#000/#FDBF2D).
> Стиль: **OLED Dark (dark-only)**. Стек: Next.js 15 + Refine + shadcn/ui + Tailwind + Framer Motion.

## 1. Цвета (бренд-токены)

| Токен | HEX | Назначение |
|---|---|---|
| `--bg` | `#000000` | фон приложения (OLED) |
| `--surface-1` | `#0D0D0D` | карточки, сайдбар |
| `--surface-2` | `#161616` | вложенные блоки, hover-строки |
| `--border` | `#262626` | границы 0.5–1px, разделители |
| `--fg` | `#FAFAFA` | основной текст (контраст ≥ 4.5:1) |
| `--fg-secondary` | `#A3A3A3` | вторичный текст (≥ 3:1) |
| `--fg-muted` | `#6B7280` | подписи, плейсхолдеры |
| `--accent` | `#FDBF2D` | CTA, активный пункт nav, фокус |
| `--accent-hover` | `#E0A800` | ховер акцента |
| `--accent-fg` | `#000000` | текст на жёлтом |
| `--success` | `#22C55E` | «Одобрить», онлайн |
| `--danger` | `#EF4444` | «Бан», жалоба, удаление |
| `--warning` | `#F59E0B` | «Мут», предупреждение |

Правила: цвет не единственный признак статуса (иконка+текст). Danger-действия визуально отделены от primary.

## 2. Типографика

- **UI/текст**: Inter (300/400/500/600/700) — уже бренд app.
- **Моно**: JetBrains Mono — для **ID (#00001)**, таймстампов, счётчиков (`tabular-nums`, без скачков ширины).
- Шкала: 12 · 13 · 14 · 16 · 20 · 28. Body 14, line-height 1.5. Заголовки 600.

## 3. Форма и эффекты

- Радиус: карточки `12px`, кнопки `10px`, чипы/бейджи — pill.
- Границы `0.5–1px` `--border` (видимы в dark).
- Тени: мягкие, низкие; на акценте — тонкое жёлтое свечение `0 0 0 3px rgba(253,191,45,.25)` для focus-ring.
- Иконки: **Lucide** (один сет, stroke 1.5–2, размеры-токены 16/20/24). Без эмодзи.

## 4. Layout (а-ля Datta Able)

- **Сайдбар** слева (`--surface-1`), сгруппированная навигация: Обзор · Пользователи · Жалобы · Медиа · Баны · Настройки. Активный пункт — жёлтый индикатор + подсветка.
- **Топбар**: поиск (ник / #ID), колокольчик уведомлений, меню профиля.
- **Контент**: верхний ряд **стат-карточек** (Всего юзеров · Онлайн сейчас · Жалоб в очереди · Банов сегодня) → таблицы/очереди ниже.
- Адаптив: ≥1024px сайдбар постоянный; <1024px — collapsible drawer. Брейкпоинты 375/768/1024/1440.

## 5. Анимации (Framer Motion) — «красиво + не мешает»

Единые токены: `fast 140ms` · `base 200ms` · `slow 320ms`. Enter = ease-out, exit = ease-in и на ~30% короче. Только `transform`/`opacity`. **Везде уважать `prefers-reduced-motion`** (отключать/сокращать).

| Элемент | Движение |
|---|---|
| Переход страниц | fade + slide-up 8px, enter 200ms / exit 140ms |
| Активный пункт nav | скользящий индикатор через `layoutId`, spring |
| Строки таблиц/списки | stagger 30–40ms/строка, FadeInUp (только видимые) |
| Ховер карточки | scale 1.02 + жёлтое свечение границы, 150ms |
| Модалка/дровер | scale 0.96→1 + fade от источника; scrim 50% чёрного; enter 200 / exit 140 |
| Нажатие кнопки/карточки | scale 0.97 spring, возврат на release |
| Тосты | slide-in, авто-скрытие 3–5с, `aria-live=polite` |
| Загрузка >300ms | skeleton-shimmer (не пустой экран/спиннер-фриз) |

Правило: 1–2 ключевые анимации на экран, не «всё прыгает». Анимации прерываемы, не блокируют ввод.

## 6. Компоненты (план)

- `AppSidebar` — сгруппированная nav + активный индикатор.
- `Topbar` — глобальный поиск (ник/#ID) + уведомления + профиль.
- `StatCard` — метрика + дельта + мини-прогресс (tabular-nums).
- `UsersTable` — колонки: Ник · **#ID** (моно) · онлайн-точка · жалобы (бейдж) · дата рег · **действия** (Одобрить/Бан инлайн). Сортировка (`aria-sort`), виртуализация при 50+.
- `ReportsQueue` — очередь жалоб: кто/на кого/причина/медиа-превью → Разобрать/Бан/Отклонить.
- `UserDetailDrawer` — профиль + **медиа-галерея** (yet-another-react-lightbox: фото+видео, свайп/зум) + **видеоплеер** (vidstack) + просмотр переписки + бан/мут/undo.
- `MediaGallery` / `MediaViewer` — лайтбокс + плеер, тёмный scrim.
- `BansList` · `LoginPage` (тёмная, жёлтый CTA).

## 7. Приёмка (из скилла)

- Контраст текста ≥4.5:1 в dark; границы/состояния видимы.
- Touch/click ≥44px; cursor-pointer; focus-ring виден для клавиатуры.
- Иконки — SVG (Lucide), не эмодзи; один сет.
- `prefers-reduced-motion` уважается; анимации transform/opacity, без CLS.
- Скелетоны на загрузку >300ms; пустые/ошибочные состояния с recovery.
- Проверка на 375/768/1024/1440.
